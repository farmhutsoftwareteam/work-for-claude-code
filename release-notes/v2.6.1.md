# Atelier 2.6.1 — skills fixes

- **Fixed: editing a skill no longer silently pins its model.** By default a
  skill just runs under whatever model your session is already using. The
  editor now has a real "inherit" option and never forces a model or effort
  level you didn't choose.
- **Fixed: pasting a link to one specific skill inside a larger repo now
  works.** Marketplaces like `anthropics/skills` bundle many skills together
  — pasting a link straight to one now clones the right branch and finds
  that exact skill instead of guessing across the whole repo.
