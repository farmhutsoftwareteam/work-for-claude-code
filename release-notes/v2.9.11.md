# Atelier 2.9.11 — window fits your screen again

- **Fixed: the window couldn't shrink small enough to fit on some
  laptop screens.** The new model/effort/permissions pill shipped
  yesterday always rendered at full width, unlike every other header
  control — that alone was pinning the whole window's minimum size. It
  now shrinks the same way the rest of the header does as the window
  narrows.
- **Removed duplicate session info from the header** — the session
  name and its live/working indicator are already on the tab, so the
  header no longer repeats them.
