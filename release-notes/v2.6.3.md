# Atelier 2.6.3 — the transcript stops bouncing

- **Fixed: scrolling a long conversation would bounce up and down.** A
  long-chat performance fix from a few releases back capped how much of the
  transcript gets laid out at once — but it kept re-centering that cap on
  every new message, so once a conversation passed ~120 messages, scrolling
  near the top of that window while a reply was streaming meant rows would
  appear and disappear under your cursor. The window is now set once when a
  conversation opens and never moves again on its own.
