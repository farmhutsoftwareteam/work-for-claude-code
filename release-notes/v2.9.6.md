# Atelier 2.9.6 — no more silent model switches

- **You now see when the API swaps your model.** When the model you
  picked is rate-limited or unavailable, the API can silently run a
  turn on a different model (e.g. Fable 5 → Opus 4.8). That used to
  render as a cryptic "[unknown block: fallback]" — one real
  conversation had seven of these invisible switches. It now shows a
  clear inline note naming both models, in live sessions and reopened
  history.
- **Pasted images render as a proper chip** ("image · png") instead of
  "[unknown block: image]".
