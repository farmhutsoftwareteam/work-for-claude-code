# Atelier 2.5.6 — projects you actually chat with, actually show up

- **Fixed: a project used only through Atelier could look untouched
  forever.** Session history was being counted from a log that Atelier's own
  chats never write to — so a project you'd used constantly could show zero
  sessions and sink to the bottom of your project list, indistinguishable
  from one you'd never opened. Session activity is now read from the real
  conversation files on disk, which every chat writes to regardless of how
  it was started.
- **Fixed: reopening a session in a project whose folder name has a space or
  punctuation could show an empty conversation**, even though the real
  history was sitting right there on disk. Atelier was guessing the wrong
  storage location for those folders; it now finds the right one.
