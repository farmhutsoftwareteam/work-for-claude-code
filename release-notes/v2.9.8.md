# Atelier 2.9.8 — MCP buttons stop feeling broken

- **Buttons now acknowledge your click.** Every chip button in the app used
  to give zero visual feedback on tap — on a slow action (marketplace
  install, connecting a server) that reads as "did this even register?"
  Buttons now dim instantly on press.
- **Installing an MCP server no longer has a dead gap.** Tapping "install"
  used to sit silently for half a second while two sheets swapped behind
  the scenes. The row now flips to "installing…" the instant it registers.
- **Deleting a server shows real progress** — a "deleting…" state on the
  row instead of it just sitting there until it silently vanishes (or
  silently doesn't, if it failed).
- **Fixed: a malformed marketplace entry could get a row stuck on
  "installing…" forever** with no way to retry. It now shows a clear error
  instead.
- **Fixed: connecting Supabase without a resolved claude binary opened a
  blank sheet.** Now shows an explanation instead.
