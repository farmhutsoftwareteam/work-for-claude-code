# Atelier 2.10.8 — Codex sub-agents, visible

- Codex tabs now show delegation cards and a live sub-agent strip, the same way Claude tabs do. When Codex spawns agents to work in parallel, each one gets a card with its name, task, live status, and elapsed time — and the strip keeps them visible after the spawn has scrolled out of view.
- Click a Codex delegation card to peek inside that agent's own work, read live from its thread.
- Multi-agent rows now name the agents they touched instead of printing raw thread IDs, in a stable order.
- Fixed "(no agent state)" — a placeholder that appeared whenever a multi-agent call didn't happen to carry status, which is normal. Those rows now say what the call actually did, like "spawned fix_backend_billing".
- Fixed the baffling "/root interacted" line: sub-agent notes now read as sentences, and "/root" is named as the main thread.
