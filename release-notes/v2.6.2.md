# Atelier 2.6.2 — cleanup and update detection for skills

- **Duplicate skill files get a real cleanup action.** If a skill ended up
  installed twice (a live copy plus a leftover archive), the skills panel
  now shows a "clean up" banner that removes the redundant copy — moved to
  Trash, always undoable.
- **Skills you've installed from a plugin now tell you when there's an
  update.** If the plugin's version has changed since you installed it,
  you'll see an "update available" badge. Nothing overwrites automatically
  — you get a side-by-side comparison and choose to keep your copy or take
  the new one.
