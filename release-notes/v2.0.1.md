# Atelier 2.0.1 — v2 actually opens this time

Hotfix on top of 2.0.0. The v2 Window scene declaration in `WorkApp.swift`
was wrapped in `#if DEBUG`, so when 2.0.0 shipped as a Release build the
`Window(id: "v2-preview")` scene didn't exist — `V2AutoOpen.openWindow(id:
"v2-preview")` silently no-op'd and Atelier kept opening into the v1
chrome.

This release drops the `#if DEBUG` guard so the v2 scene is registered
in Release too. Same code as 2.0.0 otherwise; just makes the v2 default
actually take effect.
