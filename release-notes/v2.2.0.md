# Atelier 2.2.0 — faster, with status you can hear

A big performance pass, a new tab status language with sound, and full MCP
management — plus markdown tables, one-click copy, and a layout that no longer
breaks when the window is narrow.

## Performance — long sessions feel like a different app

The chat used to slow down and stutter as a reply grew. The whole streaming
path was reworked:

- The streaming message is no longer re-parsed on every token — a long reply
  streams at **steady speed** instead of getting slower as it goes.
- Token updates are **coalesced (~30fps)** and the transcript is **lazy**, so
  scrolling stays smooth even in long conversations.
- A streamed token no longer re-renders the whole window — only the chat does.

## Tabs that tell you what's happening — and let you hear it

Tabs now speak a clear four-state status, separate from which one is selected:

- **Working** — a pulsing dot, a live elapsed timer, and a moving line under the
  tab.
- **Done · unseen** — a sage dot when a turn finishes **while you're on another
  tab**; clears the moment you look.
- **Needs you** — a glowing clay dot when a tab is **blocked on your approval**.
- A **summary in the title bar**: how many tabs are working / done / need you.
- **Sound notifications** — a gentle chime when a background turn finishes, a
  more insistent one when one needs your approval. Only for tabs you're *not*
  looking at. Toggle it from the speaker in the title bar.

## MCP, managed in the app

- See every MCP server a project is configured with — its `.mcp.json` plus your
  user/local servers — each with a **brand logo** and its **real connection
  state**.
- **Sign in to a server in one click** — the OAuth flow runs without a visible
  terminal and reconnects the session for you when it's done.
- A **red dot** flags any project (and the MCP tab) with a server that needs
  sign-in, so you can spot it at a glance.

## Transcript

- **Markdown tables now render** as actual tables instead of raw `| … |` text.
- **One-click copy** on code blocks, and a copy action under each message.

## Polish

- The session header and composer **adapt to width** — titles truncate and
  controls collapse instead of wrapping into a broken stack.
- Light / Dark / System is a **menu** now, and your **composer draft is kept
  per tab** so switching tabs never loses what you were typing.
