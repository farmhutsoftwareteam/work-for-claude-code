# Atelier 2.9.16 — MCP install/delete stop hanging

- **Fixed: installing or deleting an MCP server could leave the button
  stuck on "saving…"/"deleting…" long after it actually finished.**
  Both were waiting on a full reload of every project's config before
  clearing their loading state, instead of on the actual save/delete —
  which had already completed. The server would show up correctly if
  you backed out and checked, but the button itself looked hung the
  whole time. Both now clear the instant the real write finishes.
