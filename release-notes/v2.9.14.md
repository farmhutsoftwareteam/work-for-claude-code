# Atelier 2.9.14 — MCP panel leads with what's actually yours

- **The MCP panel now shows your project's own servers first.**
  "Available everywhere" used to list every global server with the
  same weight as the ones actually scoped to the project you're
  looking at, which made the panel noisy and hard to scan. This
  project's servers are now the primary, always-visible list;
  everywhere-servers collapse into a one-click disclosure instead of
  crowding the top. Using "→ project" on a global server still moves
  it into this project's list, same as before.
