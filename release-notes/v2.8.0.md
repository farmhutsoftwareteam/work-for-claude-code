# Atelier 2.8.0 — see your agents work, not just background tasks

- **Delegated agents are now visible.** When Claude hands work off to a
  subagent, it shows up in the transcript as a delegation card — what
  it's doing, how long it's been running, and a live one-line tail of
  its current action, instead of vanishing until it reports back.
- **Click a delegation card to watch it work.** The peek follows the
  agent's own transcript live as it runs, and shows its final report
  once it's done.
- **Co-driven terminals no longer stack up forever.** A terminal that
  finishes cleanly now folds itself to its header automatically, the
  close button is a real labeled "dismiss," and Claude can now close a
  terminal itself once it's read the output.
- **Skills: "+ new" is now a clear two-way choice** — Add with Claude
  or Add with repo — instead of only the AI-drafting flow. Pasting a
  GitHub link to add a skill is easy to find now, not buried inside
  the marketplace.
- **The skill editor is no longer painful to write in.** The SKILL.md
  body editor now fills the sheet instead of a cramped 160px box, and
  the fields you rarely touch collapse behind "more options."
- Fixed a crash that could hit on tab switch under specific conditions
  (never reached a release — caught before shipping).
