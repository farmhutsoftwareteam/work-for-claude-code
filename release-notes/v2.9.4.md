# Atelier 2.9.4 — subagent visibility, fixed for real

- **Fixed: subagent tool calls leaking into your main conversation.**
  A delegated agent's own actions — including its errors — were being
  rendered as ordinary tool widgets in the main transcript,
  indistinguishable from the assistant's own work. An agent hitting a
  transient error it self-corrects a moment later looked like an
  unexplained failure in your chat. Subagent activity now stays where
  it belongs: on its delegation card.
- **Fixed: the delegation card's peek never showed errors.** The
  card's live feed only read the agent's actions, never their results
  — a failing tool call was completely invisible even with the peek
  sheet open. Failures now show as a red "✗ reason" line.
- **Fixed: delegation cards freezing during thinking.** Agents spend
  long stretches (30–90+ seconds) reasoning between tool calls; the
  card's live line used to freeze on the last tool call the entire
  time. It now shows "‹ thinking…" so a working agent never looks
  hung.
