# Atelier 2.3.0 — files you can touch, and updates you can see

File mentions in chat are now live — peek at any file without leaving the
conversation — the agent's actions speak one visual language, and updates
finally announce themselves.

## File peek

- **Click any file the agent mentions** — `PLAN.md`, `src/poll/cursor.ts` — and
  it opens in a themed in-app preview. Markdown renders as a document, code
  gets line numbers (numbers stay out of your selection), images sit on a
  checkerboard, huge files show their head with an "open in editor" escalation,
  and binaries offer Reveal in Finder.
- Only files that **actually exist** light up — no dead links, ever.
- **Select and copy** inside the preview, or use the one-click **copy text** /
  **copy path** actions. Esc closes; the session keeps streaming behind it.

## The agent's actions, one vocabulary

- Tool calls show their target as a **token chip** (paths, globs) or an ink
  **command chip** (bash), with live status: spinner while running, **✓** on
  success, **✗** on error.
- Task lists render as a real **checklist** (done · in-progress · pending),
  errors get a proper boxed treatment, and table verdict words carry color
  (safe → sage, watch → clay).
- Links are **blue, underlined, and get a pointing-hand cursor** — and they
  open in your browser.

## Visible updates

- Atelier now **checks for updates automatically and shows the standard update
  dialog** with release notes when a new version ships — no more silent
  installs you never knew about.
- Fixed the "old version still running in the background" bug for good: the
  newest launch now retires any stale instance, closing the window quits the
  app, and the legacy terminal window no longer opens at launch.

## Quality of life

- **Models come from the CLI itself** — new models (like Sonnet 5) appear in
  the picker the day your Claude Code ships them, no history required.
- **New project** tab in Add Project: create a blank folder (optionally
  `git init`ed) and land straight in a session — no Finder detour.
- The message **copy button is always visible** — no more hover hide-and-seek.
