# Atelier 2.9.2 — "is this still doing anything?"

- **Fixed: a stalled reply looked identical to a finished one.** The
  streaming indicator used to shrink to a single small dot the instant
  the first word of a reply arrived — if generation then paused (rate
  limiting, a slow tool call, a network hiccup), there was no signal
  it was still alive. Now shows "still working… Ns since last update"
  once a reply's gone quiet for a few seconds.
- **Fixed: the header went silent while a subagent kept working.**
  Once your message got a reply, the header showed "Ready" even if a
  delegated subagent or background task it started was still running
  — the only surviving cue was an easy-to-miss dimmed dot on the tab.
  The header now shows "Working in background" until everything's
  actually done.
- **Fixed: Usage tab cost totals were under-reporting real spend.**
  Two bugs, both confirmed against real session data — 1-hour prompt
  caching was being priced at the cheaper 5-minute rate, and Claude
  Sonnet 5 had no pricing entry at all, so its usage silently didn't
  count toward your totals. Also fixed a small display bug where
  per-turn duration was truncated instead of rounded.
