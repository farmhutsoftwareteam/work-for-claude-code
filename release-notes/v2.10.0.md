# Atelier 2.10.0 — Claude and Codex, side by side

- Start either a Claude or Codex session from the V2 project home or tab bar.
- Sign in through the local Codex app-server and use the ChatGPT/Codex subscription already managed by Codex.
- Pick from the live Codex model catalog, including Sol whenever it is available to the signed-in account, with each model's supported reasoning levels.
- Resume Codex threads, stream replies into the native transcript, interrupt turns, and review command, file, permission, and MCP approval requests.
- Attach images or files, answer Codex follow-up forms, and monitor token/context usage without leaving the session.
- Inspect Codex MCP server status, complete OAuth login, and add stdio or HTTP servers to user or project-scoped Codex configuration.
- Switch an idle tab between Claude and Codex with a bounded, provider-neutral checkpoint of visible work; hidden reasoning and provider credentials never cross the boundary.
- Restore active tabs and drafts after relaunch, with lazy Codex thread resume so reopening a workspace does not start every saved runtime at once.
- Keep existing Claude sessions and Claude MCP configuration behavior unchanged.
