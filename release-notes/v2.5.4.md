# Atelier 2.5.4 — a perfectly still scrollbar

- **The scrollbar stops dancing.** 2.5.3 stopped the transcript yanking you
  to the bottom, but its scroll correction still ran after every layout pass
  — so the scrollbar visibly jittered even in an idle open chat. Scroll
  anchoring is now handled by the system *during* layout (the same mechanism
  built for chat feeds): at the bottom it follows the stream with zero scroll
  commands, scrolled up it holds perfectly still, and returning to the bottom
  re-engages tracking on its own.
- Sending a message still always jumps to your reply; resumed sessions open
  bottom-anchored.
