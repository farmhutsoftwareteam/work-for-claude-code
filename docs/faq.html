FAQ

What is Atelier?

A native Mac app that turns Claude Code into a workshop: up to six live sessions in one window, searchable chat history, and a real UI for the MCP servers, skills and plugins you'd otherwise configure by hand.

Do I need Claude Code installed?

Yes. Atelier drives the claude binary already on your machine - it's a workshop around the tool, not a replacement for it.

Does it change my config or files?

It edits the same config and files Claude Code uses, but only what you ask it to - and you can still hand-edit everything yourself. No hidden state, no separate copy of your work.

Where does my data live?

On your machine. Atelier uses your existing auth and reads/writes locally - there's no Atelier account and no cloud middleman.

Does it support agents, loops and harnesses?

Yes. Atelier gives the whole Claude Code stack a real UI: build custom agents, run goal-driven loops with a verifier and turn budget, and wrap long work in a harness - plan → work → review with a progress file and checkpoints. Hooks, skills, MCP and plugins are managed in the same place.

Windows or Linux?

macOS first. Windows and Linux builds are on the bench - they're coming.

Is it really free?

Yes - free while in active development, with no account and no license keys. See pricing.

How do I update?

Atelier checks for new builds and updates in place. Every release is listed on the changelog.

Still curious?

The fastest answer is to open the workshop.
