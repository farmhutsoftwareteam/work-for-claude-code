loops & harnesses · the new way to work

Stop prompting.
Start running loops.

Give Claude a goal it can check, a verifier to grade the work, and a turn budget so it can't run away. Atelier builds the loop, watches every turn, and stops at the finish line.

For longer work, wrap it in a harness - a plan → work → review cycle with a progress file and checkpoints, so a fresh context picks up exactly where the last left off.
